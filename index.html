Tentang Saya

Saya lahir di Negara, Bali pada tanggal 1 Februari 2002 sebagai anak ke-2. Sejak SD saya memiliki ketertarikan pada animasi setelah saya menonton Detective Conan, saya sering mengikuti lomba melukis dan menggambar walaupun tidak pernah keluar sebagai juara. Hingga saat ini saya masih menekuni hobi saya yaitu menggambar dan juga bermain game.

Saya memiliki cita-cita sebagai seorang guru dan Animator, Maka dari itu saya memberanikan diri untuk kuliah di Undiksha dan memilih prodi PTI walaupun saya tidak suka ngoding dan pengkabelan.
